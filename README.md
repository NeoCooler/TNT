Installation
-----------

    Open Terminal with (ctrl + alt + T)
    $ sudo wget https://github.com/NeoCooler/TNT/archive/master.zip
    $ sudo unzip master.zip
    $ sudo rm master.zip
    $ sudo mv TNT-master TNT
    $ sudo chmod -R 777 TNT

Usage
-----

    Open Terminal with (ctrl + alt + T)
    $ cd TNT
    $ ./run

Installation
-----------

    Open Terminal with (ctrl + alt + T)
    $ sudo apt-get install git-core
    $ git clone git://github.com//NeoCooler/TNT.git TNT

Usage
-----

    Open Terminal with (ctrl + alt + T)
    $ cd TNT
    $ ./run
